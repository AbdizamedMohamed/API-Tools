Counter API Documentation

Overview

Welcome to the Counter API documentation. This API allows you to manage counters within namespaces, offering features such as value addition, incrementation, decrementation, and more. Below, you'll find the available endpoints, methods, and their usage instructions.

Base URL

https://counter.apitools.workers.dev

Use this base URL to access the Counter API endpoints described below.

Available Endpoints

1. Get Namespace Data

Retrieve all values stored under a specified namespace.

GET /npdata?np=[NAMESPACE]&api=[API_KEY]

Example:

GET /npdata?np=myNamespace&api=yourApiKey

2. Create Namespace

Create a new namespace to store values. A unique namespace name must be provided.

POST /newnp?name=[NAMESPACE_NAME]

Example:

POST /newnp?name=myNamespace

3. Add Value to Namespace

Add a new value to a specific namespace.

POST /addvalue?np=[NAMESPACE]&api=[API_KEY]&value=[VALUE]

Example:

POST /addvalue?np=myNamespace&api=yourApiKey&value=testValue

4. Increment Value

Increment the value of a specific key within a namespace.

POST /plus?np=[NAMESPACE]&api=[API_KEY]&key=[KEY]

Example:

POST /plus?np=myNamespace&api=yourApiKey&key=myKey

5. Decrement Value

Decrement the value of a specific key within a namespace.

POST /minus?np=[NAMESPACE]&api=[API_KEY]&key=[KEY]

Example:

POST /minus?np=myNamespace&api=yourApiKey&key=myKey

6. Delete Value

Delete a specific key and its value from a namespace.

DELETE /delvalue?np=[NAMESPACE]&api=[API_KEY]&key=[KEY]

Example:

DELETE /delvalue?np=myNamespace&api=yourApiKey&key=myKey
