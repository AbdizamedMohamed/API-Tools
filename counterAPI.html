Counter API

About This API

The Counter API allows you to manage and manipulate numeric values stored in namespaces. You can create namespaces, add or remove values, and perform arithmetic operations to adjust those values dynamically.

Available API Endpoints

API Endpoints List
Endpoint Method Description
/api/ns?name=your_namespace_name POST Creates a new namespace with a specified name and auto-generated API key.
/api/add?np=your_namespace&value=your_value&api=API_KEY POST Adds a value to a namespace.
/api/set?np=your_namespace&valueName=your_value_name&newValue=your_value&api=API_KEY POST Sets or increments a value in a namespace using the value name directly.
/api/get?np=your_namespace&value=your_value&api=API_KEY GET Retrieves a value from a namespace.
/api/reset?np=your_namespace&api=API_KEY POST Resets all values in a namespace.
/api/delete?np=your_namespace&value=your_value&api=API_KEY POST Deletes a value from a namespace.
/api/deletenp?np=your_namespace&api=API_KEY POST Deletes all values and the namespace itself.
/api/plus?np=your_namespace&valueName=your_value_name&amount=your_amount&api=API_KEY POST Increases a numeric value in a namespace by a specified amount.
/api/minus?np=your_namespace&valueName=your_value_name&amount=your_amount&api=API_KEY POST Decreases a numeric value in a namespace by a specified amount.
